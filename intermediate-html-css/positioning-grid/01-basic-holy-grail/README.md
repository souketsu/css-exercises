# 使用 Grid 实现圣杯布局

现在是时候练习我们所学的 Grid 工具，创建一个看起来熟悉的布局了。我们将使用 Grid 制作一个圣杯布局，就像我们之前使用 Flexbox 创建的那样。不同之处在于，在这个练习中我们不会依赖 Flexbox，而只会使用 Grid。你将能够亲自看到 Grid 的好处！

## 提示

- 你只需要添加到 CSS 选择器中
- 如果你忘记了如何将元素转换为网格，请回顾创建网格的课程
- 对列和行使用固定的轨道大小（例如像素）
