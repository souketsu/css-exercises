# 布局圣杯

在最后这个flexbox练习中，你将重新创建一个极为常见的网站布局。它非常普遍，以至于常被称为[圣杯](https://www.google.com/search?q=holy+grail+layout&tbm=isch&sclient=img)布局... 使用flexbox，实现这个布局其实相当容易。

与上一个练习一样，我们给你留了更多需要自己完成的内容。

## 提示

- 你需要更改flex-direction来将页脚推到底部。
- 你需要添加一些div作为容器来使元素正确对齐。
- `flex-wrap` 将帮助你正确对齐卡片。
- 确保你定义了卡片应该占用的空间大小，以便 `flex-wrap` 能够按预期工作。

## 自我检查

- 页眉文本大小为32px，字重为900。
- 页眉文本垂直居中，距离屏幕边缘16px。
- 页脚被推到屏幕底部（如果'卡片'部分的内容溢出和/或你的屏幕较短，页脚可能会低于屏幕底部）。
- 页脚文本水平和垂直居中。
- 侧边栏和卡片占据页脚上方的所有可用空间。
- 侧边栏宽度为300px（且不缩小）。
- 侧边栏链接大小为24px，白色，没有下划线文本装饰。
- 侧边栏有16px的内边距。
- '卡片'部分周围有48px的内边距。
- 卡片水平排列，但当页面空间不足时会换行到多行。
