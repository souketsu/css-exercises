# 常见的 '模态框' 样式

这是网络上另一种非常常见的模式。这个练习的解决方案很_简单_...但对你来说可能不会立刻显而易见。你需要稍微编辑HTML，以使所有内容都位于正确的位置。

## 提示

根据你解决这个问题的方式，你可能需要重新学习 `flex-shrink` 属性，以防止弹性项目被挤压。此外，请注意HTML的结构，特别是考虑在header、button、main text、cancel和continue divs周围添加一个额外的容器；并考虑移动header div以同时包含button。

### 自我检查

- 蓝色图标左对齐。
- 图标的两侧有相等的空间（图标与卡片边缘之间的间隙，以及图标与文本之间的间隙是相同的）。
- 模态框边缘周围有内边距。
- 标题、文本和按钮相互对齐。
- 标题是粗体，并且文本大小比正文稍大。
- 关闭按钮与标题垂直对齐，并对齐在卡片的右上角。
